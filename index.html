👑 Owner

Nhập mật khẩu để giải mã API keys và đăng nhập

Tạo mật khẩu

Mật khẩu mã hóa API keys. Không thể khôi phục nếu quên.

API Keys

Keys sẽ được mã hóa AES-256 trước khi lưu.

Bootstrap Gist

Gist công khai lưu keys đã mã hóa, dùng để đăng nhập từ mọi thiết bị.

1. Vào gist.github.com → tạo Gist mới (nội dung bất kỳ)
2. Copy ID trong URL: gist.github.com/user/ID_Ở_ĐÂY
3. Dán vào ô dưới → Lưu → sau đó hardcode vào BOOTSTRAP_GIST_ID trong index.html và re-deploy

🎯 Guest

Nhập API keys của bạn. Dữ liệu tách biệt hoàn toàn với Owner.

Keys được lưu vào localStorage trình duyệt này. Trên thiết bị khác chỉ cần nhập lại.
